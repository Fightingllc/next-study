• 盒子模型

    盒模型

    标准盒模型：

    • width=width
    • height=height
    • 盒子总宽度 = width + padding + border + margin;
    • 盒子总高度 = height + padding + border + margin

    怪异盒模型：

    • width=width+padding+border
    • height=height+padding+border
    • 盒子总宽度 = width + margin;
    • 盒子总高度 = height + margin

    box-sizing: content-box|border-box|inherit:

    • content-box 默认值，就是标准和模型
    • border-box，就是怪异盒模型
    • inherit，继承

  • 伪类&伪元素

    • 伪类：active hover focus link first-child
    • 伪元素：before after
    • 继承：字体系列、文本系列
    • no extend:布局flex 定位 盒模型属性
